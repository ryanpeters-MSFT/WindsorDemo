# Windsor IoC Example with .NET 6/7

This example integrates a custom `IServiceProviderFactory<>` instance to build a service provider. It also uses instances of `IWindsorInstaller` classes to register components. 

Packages used:
- Castle.Windsor
- Castle.Windsor.MsDependencyInjection

_This code is not fully tested and is to be used at your own risk. I assume no responsibility for its use._
